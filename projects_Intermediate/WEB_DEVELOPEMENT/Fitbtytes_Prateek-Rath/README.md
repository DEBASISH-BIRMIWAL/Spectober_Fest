## Tech Stack
`Javascript` `ReactJS` `Material-UI`

## Description
Fitbytes is a work from home exercise and fitness application where user can search from more than 1000 different exercises and get detailed steps, train a specific body part, get professional videos related to the exercise and similar exercises all at one place.
